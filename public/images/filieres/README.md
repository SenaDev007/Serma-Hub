# Images des filières (section « Les 5 filières »)

Place dans ce dossier une image par filière, au format **JPG**, avec le nom exact suivant :

| Fichier | Filière |
|---------|---------|
| `commerce-distribution.jpg` | Commerce & Distribution |
| `agro-business.jpg` | Agro-Business & Transformation |
| `services-techniques.jpg` | Services Techniques |
| `digital-local.jpg` | Digital Local |
| `entrepreneur-feminin.jpg` | Entrepreneuriat Féminin |

**Recommandations :**
- Format : JPG (ou modifier l’extension dans le code si tu utilises PNG/WebP)
- Ratio conseillé : 16/10 (ex. 600×360 px ou 800×500 px)
- Contenu : visuel en lien avec la filière (commerce, agriculture, technique, digital, entrepreneuriat féminin)

Si un fichier est absent, un fond coloré (couleur de la filière) s’affiche à la place.
